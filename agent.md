# 🤖 AI Agent: The Auditor

## 🎯 Role & Objective
The AI Agent in this system serves as a **Specialized Quality Assurance Auditor**. Its primary goal is to ensure that medical/service session notes accurately reflect the actual phone calls between staff and clients.

## 🧠 Core Competencies

### 1. Temporal Analysis (Start/End Times)
- **Logic**: Compares the `Session Start Time` against the `Call Connection Time`.
- **Threshold**: If the session creation is more than 20 minutes apart from the actual call, it flags the discrepancy.
- **Goal**: Prevent fraudulent or inaccurate time entries.

### 2. Billing Compliance (Overbilling Detection)
- **Logic**: Calculates the duration of the call transcript and compares it to the `Billed Units`.
- **Threshold**: If billed units exceed the call duration by more than a standard margin, it marks it as "Overbilled".

### 3. Transcript Intelligence
- **Entity Matching**: Ensures the staff member listed is the one actually speaking.
- **Goal Analysis**: Verifies if the service goals mentioned in the notes were actually discussed during the call.
- **Sentiment & Tone**: (Optional) Can detect frustrated or non-compliant interactions.

## 🛠 Model Configuration
- **Engine**: `gpt-4o-2024-08-06`
- **Output Mode**: `Structured Outputs` (JSON Schema)
- **Reasoning Depth**: High - provides detailed "Reasons" for every flag generated.

## 🔄 Interaction Loop
1. **Input**: Cleaned JSON containing session notes + call transcripts.
2. **Analysis**: Runs a multi-stage prompt that acts as "Start Time Checker", "End Time Checker", and "Service Quality Auditor" simultaneously.
3. **Decision**: Assigns a `Good`, `Flagged`, or `Overbilled` status.
4. **Action**: Returns a structured response that is mapped directly to Monday.com columns.

## 🛤 Future Roadmap
- [ ] **Proactive Intervention**: Alerts supervisors via Slack/Email the moment a flag occurs.
- [ ] **Cross-Session Correlation**: Detect patterns of inconsistent reporting across multiple days for a single staff member.
- [ ] **Automated Coaching**: Suggests improvements to staff members based on the "Reasons" generated by the AI.
